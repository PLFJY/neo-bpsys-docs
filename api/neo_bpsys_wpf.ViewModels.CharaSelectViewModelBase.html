Class CharaSelectViewModelBase

Namespace
neo_bpsys_wpf.ViewModels
Assembly
neo-bpsys-wpf.dll

用于选择角色的角色选择器行为的基类 需要派生类所做的是:
1.实现 CharaDict 更新的行为
2.设置 IsEnabled 同步 ISharedDataService 的哪一个 CanCurrentBannedList 的值，通常需要搭配订阅 Ban 位数量变动的事件
3.实现 SyncCharaToSourceAsync() 将角色同步到前台的行为 4.实现 SyncIsEnabled() 通过toggle button设置后同步状态到对应的 ISharedDataService 中 CanCurrentBannedList 的值的行为 5.实现 IsActionNameCorrect(GameAction?) 判断当前步骤引导的步骤是否符合当前控件的行为 6.在 OnCurrentGameChanged(object, EventArgs) 中更新 preview 的 image
